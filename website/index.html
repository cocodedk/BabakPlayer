Android companion app for BabakCast

Play split media like one continuous story.

BabakPlayer imports shared media parts from messaging apps, preserves receive order, and runs a seamless playlist across audio and video.

Download APK View GitHub

BabakCast -> BabakPlayer

1. BabakCast creates parts

Large media gets split into shareable segments for messaging workflows.

2. Share to BabakPlayer

Send one or many files directly from WhatsApp or the Android share sheet.

3. Continuous playback

BabakPlayer keeps exact import order and auto-plays the full sequence.

Optimized for imports generated by BabakCast.

Why BabakPlayer

Strict import order

No filename resorting. Playback sequence follows exact share payload order.

Continuous queue

Press play once and continue through all parts with smooth controls.

Hard-delete controls

Delete a single file or full playlist with permanent local purge behavior.

Local-first privacy

No cloud account required. Files are stored in app-private Android storage.

Install on Android

1

Download the APK

Tap the download button above or get it from GitHub Releases.

2

Allow installation

When prompted, enable "Install from unknown sources" for your browser or file manager.

3

Install the app

Open the downloaded .apk file and tap "Install".

4

Open BabakPlayer

Launch from your app drawer and start importing media.
